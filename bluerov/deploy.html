Deploy Klopsi

SSH Access

For convenience, copy your ssh-key to the BlueROV to enable passwordless login and create an entry in your ~/.ssh/config for the BlueROV similiar to:

Host klopsi-main
    User pi
    Hostname klopsi-main.local
    IdentitiyFile "~/.ssh/id_ed25519"

Host klopsi-buddy
    User pi
    Hostname klopsi-buddy.local
    IdentitiyFile "~/.ssh/id_ed25519"

Note

Replace the path for the identity file with the name of your key.

This entry allows you to use ssh klopsi-main instead of ssh pi@klopsi-main.local.

Environment Variables Checklist

  1. Change ROS_MASTER_URI in ~/.zshrc to your desired value for both Raspberry Pis. For example .. code-block:

    ROS_MASTER_URI="http://hippo-celsius.local:11311"

    Attention

    Do not forget to resource ~/.zshrc!

  2. Change QGC_HOST in ~/bin/mavros on klopsi-main to the hostname of the computer you want to have QGroundControl running on.

    QGC_HOST="hippo-celsius.local"

Start the usual Setup

  1. Start MAVROS .. code-block:

    mavros bluerov

  2. Start

    • Front Camera

    • Lights

    • Camera Servo

    • Barometer

    roslaunch hardware_interfaces top_hardware_bluerov.launch vehicle_name:=bluerov

    All these nodes can be reconfigured via dynamic_reconfigure.

Start the vertical camera
roslaunch camera camera_node.launch vehicle_name:=bluerov

This camera can be configured via dynamic_reconfigure.
