Afro ESCÂ¶

  1. Download AVRA.

    git clone https://github.com/Ro5bert/avra.git

  2. Build and install AVRA

    cd avra && sudo make install

  3. Download the firmware

    git clone https://github.com/sim-/tgy.git

  4. Modify at least the following lines in tgy.asm

    RC_PULS_REVERSE = 1 ; This enables forward/reverse throttle
RC_CALIBRATION = 0 ;
STOP_RC_PULS = 1000 ; minimum pwm value
FULL_RC_PULS = 2000 ; maximum pwm value

  5. Optionally fiddle with the deadband

    RCP_DEADBAND = 50 ; default value

  6. Build the firmware

    make all

  7. Connect the Afro Programmer with your computer and connect the data wires with the ESC. Supply the ESC with an appropriate voltage (probably ~10-12V).

  8. Download KKmulticopter Flash Tool.

  9. Flash the afro_nfet.hex firmware you have built before. Select afrousb as programmer and leave the checkbox for default settings selected.

    Important

    Make sure avrdude verifies the flashed data successfully. Otherwise rebuilt the firmware with make clean && make all and try again.
