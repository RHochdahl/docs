control.pid.Controller¶

class control.pid.Controller(p_gain=1.0, i_gain=0.0, d_gain=0.0)¶

Bases: object

A quite normal PID controller.

PID gains can either be set on the creation of a controller instance or via the respective properties any time later on. Compute the control output by invoking update.

Examples

::

controller = Controller(p_gain=3.4, i_gain=1.1, d_gain=0.1) controller.p_gain = 1.0 print(“Controller’s p_gain: {}”.format(controller.p_gain))

control_error = 1.0 control_output = controller.update(error=control_error, dt=0.02) print(“Control output: {}”.format(control_output))
